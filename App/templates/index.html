SimplifiedText: 

Text simplification project

This project saw the light of day during the Software project course of Nancy NLP Master.

The goal was to create a rule-based text simplification system. This method present some limitation (notably taking into account sentences context). Nevertheless, the result produced if studied further could be interesting.

Our system is composed of multiple modules:

  • Syntactic simplification
  • Lexical simplification
